Welcome to the keen-pbr documentation.

keen-pbr helps you send selected sites and services through a VPN while the rest of your traffic keeps using the normal connection.

**New here?** Start with [Getting Started](getting-started/) for package installation and first setup, or jump straight to [Quick Start](getting-started/quick-start/) if keen-pbr is already installed.

For the full configuration reference, see [Configuration](configuration/).

If you want to understand how keen-pbr works under the hood, see [Concepts](concepts/).

If you need command-line tools, API details, or service signals, see [Advanced](advanced/).

If you are building packages or compiling from source, see [Developer](developer/).
