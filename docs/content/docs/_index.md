Welcome to the keen-pbr documentation.

keen-pbr helps you send selected sites and services through a VPN while the rest of your traffic keeps using the normal connection.

**New here?** Start with [Getting Started]({{< relref "/docs/getting-started" >}}) for package installation and first setup, or jump straight to [Quick Start]({{< relref "/docs/getting-started/quick-start" >}}) if keen-pbr is already installed.

For the full configuration reference, see [Configuration]({{< relref "/docs/configuration" >}}).

If you want to understand how keen-pbr works under the hood, see [Concepts]({{< relref "/docs/concepts" >}}).

If you need command-line tools or service signals, see [CLI]({{< relref "/docs/cli" >}}). For HTTP integration, see [REST API]({{< relref "/docs/rest-api" >}}).

If something is not working yet, see [Troubleshooting]({{< relref "/docs/troubleshooting" >}}) or [FAQ]({{< relref "/docs/faq" >}}).

If you are building packages or compiling from source, see [Developer]({{< relref "/docs/developer" >}}).

If you want to help keep keen-pbr moving, see [Support This Project]({{< relref "/docs/support-this-project" >}}).
