Welcome to the keen-pbr3 documentation.

keen-pbr3 is a policy-based routing daemon for Linux systems (OpenWRT, Keenetic routers, and similar). It manages firewall marks, routing tables, and ip rules to selectively route traffic from specific domain lists or IP ranges through designated outbound connections.

**New here?** Start with [Getting Started](getting-started/) to build and deploy keen-pbr3, or jump straight to the [Quick Start](getting-started/quick-start/) guide for a minimal working configuration.

For a conceptual overview of how the pieces fit together, see [Concepts](concepts/).

For the full configuration reference, see [Configuration](configuration/).
