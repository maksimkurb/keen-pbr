Добро пожаловать в документацию keen-pbr.

keen-pbr помогает направить выбранные сайты и сервисы через VPN, сохраняя обычное соединение для остального трафика.

**Впервые здесь?** Начните с раздела [Начало работы](getting-started/) для установки пакета и начальной настройки, или перейдите сразу к [Быстрому старту](getting-started/quick-start/), если keen-pbr уже установлен.

Полный справочник по конфигурации см. в разделе [Конфигурация](configuration/).

Если хотите понять, как работает keen-pbr «под капотом», см. раздел [Концепции](concepts/).

Если нужны инструменты командной строки или сигналы сервиса, см. раздел [CLI](cli/). Для HTTP-интеграции см. раздел [REST API](rest-api/).

Если что-то ещё не работает, см. раздел [Устранение неполадок](troubleshooting/) или [FAQ](faq/).

Если вы собираете пакеты или компилируете из исходного кода, см. раздел [Разработка](developer/).

Если хотите помочь развитию keen-pbr, см. раздел [Поддержать проект](support-this-project/).
