# Keenetic / NetCraze

keen-pbr можно установить на роутерах Keenetic / NetCraze через менеджер пакетов Entware `opkg`.

## 1. Установите Entware на роутере

Сначала убедитесь, что Entware установлена на вашем роутере. Пожалуйста, обратитесь к официальному мануалу роутера [для Keenetic](https://support.keenetic.com/?lang=ru) или [NetCraze](https://support.netcraze.ru/?lang=ru):

1. Найдите мануал вашего роутера.
2. В поле поиска найдите "Установка репозитория Entware на USB-накопитель".
3. Внимательно прочитайте и следуйте инструкциям.

## 2. Установите дополнительные компоненты

Откройте страницу конфигурации роутера, перейдите в **Управление** -> **Системные настройки** и установите эти дополнительные компоненты:

- Сетевые функции / **Протокол IPv6**
- Пакеты OPKG / **Поддержка системы пакетов**
- Пакеты OPKG / **Модули ядра подсистемы Netfilter**
- Пакеты OPKG / **Пакет расширений Xtables-addons для Netfilter**

## 3. Установите из страницы репозитория

Откройте страницу инструкций репозитория, выберите **Keenetic / NetCraze** в селекторе ОС слева и используйте сгенерированные команды:

{{< hextra/hero-button text="Репозиторий keen-pbr" link="https://repo.keen-pbr.fyi/repository/stable" >}}

Пример команды установки:

```bash {filename="bash"}
opkg install keen-pbr

# или если нужна версия без API и без веб-интерфейса
# opkg install keen-pbr-headless
```

Пакет устанавливает конфигурацию в `/opt/etc/keen-pbr/config.json` и автоматически включает init-скрипт.

Запустите сервис:

```bash {filename="bash"}
/opt/etc/init.d/S80keen-pbr start
```

Перезапустите после редактирования конфига:

```bash {filename="bash"}
/opt/etc/init.d/S80keen-pbr restart
```

{{< callout type="info" >}}
Если вы не планируете использовать веб-интерфейс keen-pbr или API, можно установить пакет `keen-pbr-headless`.
Он занимает меньше места (~1.2 МБ вместо ~2.8 МБ) и не включает API-сервер. Также вы можете отключить API-сервер через флаг конфигурации в любой момент в полной версии пакета.
{{< /callout >}}

Следующий шаг: откройте [Быстрый старт](../quick-start/) и используйте вкладку **Веб-интерфейс** для самой простой первоначальной настройки. Если вы установили `keen-pbr-headless`, используйте вкладку **JSON / CLI**.
